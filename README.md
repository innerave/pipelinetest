# pipelinetest


# Cобирать:
```sh
make
```
# Использовать:
```sh
pipetest файл1 файл2
```
# Ограничения
-вывод в консоль
-поскольку используется XOR вывод может состоять из специальных символов (не отображаться)
-лучше использовать перенаправление в файл оперетором ">"
-файлы должны беть одного размера
